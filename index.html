AI Advisor — Groq API key
The AI Advisor uses Groq's free API. Get your free key at console.groq.com (takes 2 min, no credit card needed).
AI Autopilot — Claude API key (optional)
The AI Autopilot works fully without any key — all insights are computed on-device. Adding a Claude key (from console.anthropic.com) unlocks richer natural-language briefings that narrate your insights. If left blank, briefings fall back to your Groq key.
Market prices — FMP API key (optional)
Optional. Get a free key at financialmodelingprep.com (250 requests/day).

What this gives you: end-of-day close prices (yesterday's close) for US-listed holdings (VOO, VYM, VYMI). This is not real-time — the FMP free plan no longer includes real-time quotes. ASX-listed ETFs (VAF, VGB, VGS) aren't covered on the free plan either — use screenshot scan or manual entry for those.
Exchange rate (USD → AUD)
Your US holdings (VOO, VYM, VYMI) are priced in USD and converted to AUD using this rate. It updates automatically when you refresh prices with an FMP key, or set it manually here (check Google for "USD to AUD").
